# Cislunar-Orbit-Determination-and-Stationkeeping
Documentation of research projects related to estimation and control problems for spacecraft operating in cislunar space. 
## Applied Methods
- Principal Stretching Direction Control (PSDC) and Floquet Mode Control (FMC)
- Extended Kalman Filter (EKF)
- Batch Least-Squares Estimation
- Numerical integration 
- Monte Carlo simulations
## Features
- Nonlinear dynamics in Earth-Moon system (CR3BP)
- Design of orbit maintenance maneuvers (omm) along halo orbits using Principal Stretching Direction and Floquet Mode control approaches
- Stationkeeping simulation analysis
- State Estimation using batch nonlinear least squares for initial orbit determination and EKF for sequential estimation
- range and range-rate measurement model
- Sensor noise modeling filter convergence evaluation through monte carlo simulations
## Tools
- Python (NumPy, SciPy, Matplotlib)

## Results
- Demonstrated stability of stationkeeping strategies for long-term propagations
- Demonstrated convergence of batch least squares and EKF under noisy measurements
- Compared batch vs EKF estimation performance
- Evaluated sensitivity to initial state errors
